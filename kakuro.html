Rules of Play

Reading clue cells the bottom-left number gives the total for the horizontal row run extending to the right, while the top-right number gives the total for the vertical column run extending downward.

Across and down clues define the exact sum for the run touching that black clue cell.

No repeated digit is allowed within a single run, even when the sum would still match.

Victory comes only when every white cell is filled and every run balances perfectly.

Keyboard support: digits 1–9 enter values, Backspace/Delete clears, and arrow keys move selection.
